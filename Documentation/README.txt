DOCUMENTACIÓN
📚 En este apartado se encuentra la documentación del proyecto, un conjunto de textos, guías y materiale que explican 
cómo fue el proceso de construccion de la estructura (caja fuerte), cómo se usa, y caracteristicas escenciales.

📔 Este proyecto propone el diseño e implementación de un sistema embebido para la gestión y control de una caja fuerte, 
combinando componentes de hardware y software para crear una solución eficiente, segura y accesible. 
Mediante el uso de microcontroladores, sensores, interfaces electrónicas y algoritmos de autenticación, 
la caja fuerte será capaz de ofrecer características avanzadas como autenticación multifactor, monitoreo remoto, 
optimizando la experiencia del usuario y reforzando la seguridad.

✏️A lo largo del documento, se describirá el desarrollo del proyecto desde su concepción hasta su implementación. 

Se abordarán aspectos clave como:
- La selección de tecnologías y componentes de hardware.
- El diseño y programación del sistema embebido.
- Las funcionalidades implementadas y las ventajas respecto a soluciones tradicionales.

Con este enfoque, se busca demostrar cómo los sistemas embebidos pueden transformar un dispositivo convencional en una herramienta inteligente, 
robusta y alineada con las demandas de seguridad del siglo XXI.
